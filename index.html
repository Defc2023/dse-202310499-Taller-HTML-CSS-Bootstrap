Listas Enlazadas

Introducción

Una lista enlazada es otra estructura de datos que, al igual que un arreglo, almacena elementos en una secuencia ordenada. Sin embargo, existen algunas diferencias clave.

Diferencias Principales

La principal diferencia es que las listas enlazadas están compuestas por objetos llamados ListNode. Este objeto contiene dos atributos:

value:
Almacena el valor del nodo. Puede ser un carácter, un entero, etc.
next:
Almacena la referencia al siguiente nodo en la lista enlazada.

La imagen a continuación visualizaría el objeto ListNode (esto se entenderá mejor más adelante).

Creando una Lista Enlazada desde Cero

Mediante el encadenamiento de estos objetos ListNode se puede construir una lista enlazada. Comenzamos definiendo la clase ListNode: 

class ListNode:
    def __init__(self, val):
        self.val = val
        self.next = None

Utilizando el puntero next de cada nodo, podemos conectar los nodos entre sí. Supongamos que tenemos tres objetos: ListNode1, ListNode2 y ListNode3. 

ListNode1.next = ListNode2
ListNode2.next = ListNode3
ListNode3.next = None

Recorrido

Para recorrer una lista enlazada desde el inicio hasta el final, se puede utilizar un bucle while sencillo: 

cur = ListNode1
while cur:
    cur = cur.next
  1. Comenzamos el recorrido en la cabeza de la lista, que es ListNode1.
  2. Asignamos este nodo a una variable cur, que denota el nodo actual.
  3. Ejecutamos el bucle while hasta llegar al final de la lista, donde cur es null.
  4. En cada iteración, actualizamos cur para que apunte al siguiente nodo en la lista (cur = cur.next).
  5. El recorrido se realiza en tiempo O(n), donde n es el número de nodos en la lista.

Lista Enlazada Circular

Un escenario interesante se presenta si el puntero next de ListNode3 se establece a ListNode1 en lugar de a null. Esto da como resultado una lista enlazada circular.

Intentar iterar a través de una lista enlazada circular resultaría en un bucle infinito, ya que nunca se alcanzaría el final de la lista.
